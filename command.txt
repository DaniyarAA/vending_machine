Какие команды я использовал для системы контроля версий:

Переключение на ветку home-work-25 и обновление её до последних изменений с удаленного репозитория:
git checkout home-work-25
git pull origin home-work-25

Просмотр изменений, которые не были добавлены в индекс:
git status

Добавление изменений в индекс:
git add .

Создание коммита с добавленными изменениями:
git commit -m "Message"

Отправка изменений:
git push
